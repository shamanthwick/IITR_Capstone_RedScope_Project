## Project

RedScope: AI-Assisted Penetration Test on a Vulnerable Lab.

## Required Scope

- DVWA
- OWASP Juice Shop
- Metasploitable-style host
- AI assistant usage with review and justification
- Small adversarial ML experiment using Python and scikit-learn

## Required Methodology

- Rules of Engagement
- Reconnaissance
- Scanning
- Vulnerability assessment
- Web exploitation
- Network/service exploitation
- Post-exploitation enumeration and simulated data movement
- ATT&CK-tagged attack timeline
- Executive and technical reporting

## Weekly Deliverables

| Week | Deliverable |
| --- | --- |
| Week 1 | Signed RoE, recon report, initial finding list |
| Week 2 | Two reproducible PoC exploits, attack-narrative timeline |
| Week 3 | ML defender, evasion write-up, executive report, technical report, debrief deck |

## Individual Scope

- 1 adversarial ML evasion technique
